# SalesLoft

This is a small application that uses SalesLoft API

### Requirements
- ruby 2.6.6
- rails 6.0.2
- node >= v10.0.0

### Installation

```
bundle install
yarn install

cp .env.example .env

# Set your own API_KEY in .env
```

### Development

```
rails s
```

### System tests
```
rails test:system
```
### Frontend tests

```
yarn test
```
